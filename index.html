Python 标识符

在python中,标识符由字母,数字,下划线组成.所有的标识符都可以使用这些.但是不能以数字开头.使用下划线(_)开头有其他意义,对大小写敏感.(及不建议使用中文创建变量,函数等,毕竟是外国的东西嘛)

以下划线开头的标识符是有特殊意义的。以单下划线开头（_foo）的代表不能直接访问的类属性，需通过类提供的接口进行访问，不能用“from xxx import *”而导入；以双下划线开头的（foo）代表类的私有成员；以双下划线开头和结尾的（foo__）代表python里特殊方法专用的标识，如init（）代表类的构造函数。
